Argus
=====

Argus is easy-to-use flexible library for training neural networks in PyTorch.

Features
--------
* Argus makes the learning process more straightforward and the code briefer.
* Simplifies the experiments with different architectures.

Project info
------------
* GitHub repository: https://github.com/lRomul/argus
* License: MIT

Installation
------------

You can use pip to install argus:

.. code:: bash

    pip install pytorch-argus

If you want to get the latest version of the code before it is released on PyPI you can install the library from GitHub:

.. code:: bash

    pip install -U git+https://github.com/lRomul/argus
